Post-Installation Testing

Initial Static Tests

Reconnect Battery

Safely reconnect battery; prevent accidental cranking.

Start Engine (Idle Only)

Start engine, idle only; no revving/load.

Check for Leaks

Check for fluid/air leaks in turbo/actuator area.

Monitor for Unusual Noises

Monitor for abnormal sounds (whining, grinding, hissing).

Verify No Fault Codes

Verify no fault codes with diagnostic scanner.

Important Notes

  • Complete these tests with the engine at idle speed, without load
  • Document any abnormal sounds or leaks for further investigation
  • Compare idle speed and noise to pre-installation levels

Next Steps

If all tests pass: Proceed to Road Test
If issues detected: Reassess installation
